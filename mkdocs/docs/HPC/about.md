# About docs.hpc

This document explains what the purpose is of this site and how it is build.  

**You -- the visitor of this site -- are invited to give feedback.**

This can be done by making an issue in our git repo or
by mailing to docs@hpc.ugent.be

We keep a log of changes to the site in the [changelog](changelog.md).
Details can be found in the history of the git repository.

In the past the documentaion was made in Latex and was available in pdf format.  
Later this was changed to markdown.  
The markdown is converted to html by a static site generator (SSG) called
'mkdocs'.  
Currently a switch is made to 'zensical', another SSG.
